
# macOS HIDS, RGC Engine & Streamlit Dashboard

This repository contains:
1. A **macOS Host-based Intrusion Detection System (HIDS)** to monitor file changes and detect system anomalies.
2. An **RGC Engine** with governance mapping for risk compliance monitoring.
3. A **Streamlit dashboard** to visualize system and risk status in real-time.

## Installation

Clone the repository and install dependencies:
```bash
git clone https://github.com/dinanazizi/macos-hids-rgc-dashboard.git
cd macos-hids-rgc-dashboard
pip install -r requirements.txt
